# Job Bank App

> A smart job-matching platform connecting job seekers and employers —  
> built end-to-end with Flutter and a PHP REST API backend.

## Features

- **Smart job matching** — Personalized recommendations based on user profile
- **Auto application** — Automated job application submission
- **Paid CV submission** — Employers submit CVs through a paid system
- **Full-stack ownership** — Database, API, and UI built solo end-to-end

## Tech Stack

| Layer    | Technology     |
|----------|----------------|
| Frontend | Flutter (Dart) |
| Backend  | PHP            |
| Database | MySQL          |
| API      | REST           |

## Architecture

Flutter mobile front-end → REST API (PHP) → MySQL  
Sole developer — independently handled database design,  
API architecture, and full UI.
